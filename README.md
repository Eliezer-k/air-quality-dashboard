Air Quality Dashboard

Project Overview

This project focuses on analyzing and visualizing air quality data using Python and Streamlit.  
The main goal is to understand pollution levels over time and provide an interactive dashboard to explore Air Quality Index (AQI) trends.

---
Objective

The main objectives of this project are:

- Analyze air quality data from different stations
- Identify pollution trends and peak pollution periods
- Visualize AQI variations over time
- Build an interactive dashboard for data exploration

---

Tools & Technologies

- Python 
- Pandas (Data Cleaning & Analysis)
- Plotly (Data Visualization)
- Streamlit (Interactive Web Dashboard)

---

Dataset

The dataset contains air quality measurements collected over time from multiple stations.

It includes:
- Pollutant levels
- Time-based records
- Station-wise data

---
Key Features

- Time-series analysis of air pollution
- Visualization of AQI variations
- Interactive charts using Plotly
- Web-based dashboard using Streamlit
- Data filtering and exploration

---

Key Insights

From the analysis, we can observe:

- Certain time periods show higher pollution levels
- Air quality varies significantly across time
- Some stations consistently record higher pollution values
